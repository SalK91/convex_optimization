## Setting Up a Python Virtual Environment

### 1. Create a Separate Python Virtual Environment
Use the following command to create a virtual environment named `.venv`:

```bash
python -m venv .venv
```

### 2. Activate the Virtual Environment

- **On Windows (Git Bash / PowerShell):**
  ```bash
  source .venv/Scripts/activate
  ```

- **On macOS / Linux:**
  ```bash
  source .venv/bin/activate
  ```

Once activated, your terminal prompt will change toindicate that the virtual environment is active.
